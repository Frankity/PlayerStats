PlayerStats
===========

A Hexchat plugin in python to get the info of the player of the Rome Project :)

#Requieriments
* python 2.7.
* HexChat with python plugin's interface activated.
* IQ over 60.

#Usage:

just tipe 

/player playername

example:

/player Frankity

output:  Frankity - Rank: 13 Score: 194472 Kills: 792 Deaths: 935
